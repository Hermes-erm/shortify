## Run Locally

Clone the project

```bash
  git clone https://github.com/Hermes-erm/shortify
```

Go to the frontend project directory

```bash
  cd shortify/client
```

Install dependencies

```bash
  npm install
  npm run dev
```

Go to the backend project directory

```bash
  cd shortify/server
```

Install dependencies

```bash
  npm install
```


Start the server

```bash
  npm start
```
